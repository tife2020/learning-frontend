After these values are set, the positions of the element can then be adjust using the CSS propeerties: top,right,bottom,left. These properties will not work on position: static

position: static

static: default position for every html element. hence, top,right,bottom,left have no effect on it

position: fixed

fixed: lifts the element and fixes it to a specified position

position: sticky

sticky: mantains a normal position but when scrolling past its normal position, the element lifts and sticks to the specified position. Behaves like a relative then fixed

position: relative

relative: maintains its normal position and can be adjusted using top, right, bottom, left
