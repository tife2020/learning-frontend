Transform

It is a css property used to change the form of an object. it has the nice the following values

  • scale

    • This scale things to be bigger or smaller. scale the left and right with transform: scaleX(), top and bottom with transform: scaleY() or all four at once transform: scale(X,Y) or scale(XY)

  • translate

    • This move things things up with e.g transform: translateY(-100px), right with transform: translateX(100px), bottom with transform: translateY(100px), left with transform: translateX(-100px), or diagonally withtransform: translate(X,Y)

  • rotate

    • This rotates the item in the degree specified with transform: rotate(90deg)
      • +

  • skew

    • tilt the object in the X-axis 0r Y-axis with transform: skew(X,Y)
